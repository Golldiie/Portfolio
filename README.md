# Portfolio

Персональное портфолио frontend-разработчика с информацией обо мне, моих проектах и навыках. Проект полностью разработан с нуля: от проектирования интерфейса до реализации интерактивности.

## Стек

- HTML
- CSS
- JavaScript
- Gulp

## Возможности

- резиновая вёрстка;
- модальные окна проектов;
- динамическое заполнение контента;
- плавные анимации;
- интерактивные элементы;
- доступная и семантичная разметка.

## Запуск

npm install
npm start
